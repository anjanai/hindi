नाम: --------------------------------------------------

Follow the instructions for each of the following words:

  1. Take a RED crayon/marker in your hand. If the word has a vowel ( अ इ ए ई आ ) , draw a RED oval around it.
  2. Now, take a BLUE crayon/marker. If the word has a half-letter, draw a BLUE oval around it.
  3. Now, PINK. If it has an ा matra, draw a PINK rectangle around it.
  4. Now, PURPLE. If it has an ि matra, draw a PURPLE rectangle around it.
  5. Now, ORANGE. If it has an ी matra, draw an ORANGE rectangle around it.
  6. Now, GREEN. If it has an े matra, draw a GREEN rectangle around it.
Note that most words will have multiple ovals and/or rectangles around them.
