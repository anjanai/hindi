Nouns and their possesive cases

In the previous class, we did the word-endings for the nouns that come AFTER the का/की/के.

This time, we did the word-endings for the nouns that come BEFORE the का/की/के.

There are simple rules when it comes to changing the noun BEFORE the का/की/के:

  • If the word is a पुल्लिंग एकवचन ending in ा, change the ा to े.
  • For all other एकवचन words (all स्त्रीलिंग and all OTHER पुल्लिंग) words, there is no change.
  • All बहुवचन words end in ओं or ों.

The following table gives examples to show you the rules.
पुल्लिंग
एकवचन ...changes to... बहुवचन ... changes to ....
कुत्ता कुत्ते कुत्ते कुत्तों
घर घर घर घरों
उल्लू उल्लू उल्लू उल्लु्ओं
स्त्रीलिंग
एकवचन ...changes to... बहुवचन ... changes to ....
बिल्ली बिल्ली बिल्लियाँ बिल्लियों
गाय गाय गायें गायों
वस्तु वस्तु वस्तुएँ वस्तुओं
लता लता लताएँ लताओं
गुड़िया गुड़िया गुड़ियाँ गुड़ियों

Exercises

Do the translation in the following steps.

  1. First find the Hindi translation for the first word.
  2. Find out if it is पुल्लिंग or स्त्रीलिंग.
  3. Find out if you need एकवचन or बहुवचन for the first word.
  4. Change the ending of the first word according to rules and/or table above.
  5. Then find the Hindi translation for the second word.
  6. Find out if it is पुल्लिंग or स्त्रीलिंग.
  7. Change ending for बहुवचन if needed.
  8. Put का/की/के depending on the second word.

Translate the following phrases into Hindi. Use the words in the box if needed.

(You will need to convert एकवचन to बहुवचन where necessary.)
eye:आँख(स्त्री) horse:घोड़ा(पु) tummy:पेट(पु) house:घर(पु) brother:भाई(पु)
book:पुस्तक(स्त्री) finger:उंगली(स्त्री) mother:माँ(स्त्री) girl:लड़की(स्त्री) shoe:जूता(पु)

    Example

    girls' shoes (shoes that belong to the girls)
    1. girl = लड़की
    2. लड़की is स्त्रीलिंग
    3. I need बहुवचन of स्त्रीलिंग word लड़की.
    4. For this kind of word, the end of the word should be changed यों/ों/ओं => लड़कियों.
    5. shoe = जूता
    6. जूता is पुल्लिंग.
    7. shoes is plural, so I need बहुवचन for the word : जूते.
    8. For पुल्लिंग बहुवचन, I need to use के. ANSWER: लड़कियों के जूते

    I hope all of you know this, but let me make it clear in any case:

    • girl's means you are talking about one girl
    • girls' means you are talking about many girls.
